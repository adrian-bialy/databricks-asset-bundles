# databricks-asset-bundles
Databricks Asset Bundle Demo

1. ./azure/deploy.sh
2. Login to Azure
3. Wait and login to Databricks
4. Enable serverless in admin console (https://accounts.azuredatabricks.net)
5. Init bundle
6. Poetry install
7. poetry run python -m demo.main --catalog dabwk13-dbx --schema adrian_bialy
8. Host/Cluster variables
